# Local Tools

Place repo-local binaries here when you want the analysis pipeline to prefer them over vendored or system tools.

Suggested structure:

- `analysis/tools/local/renderdoc/renderdoccmd`
- `analysis/tools/local/fossilize/fossilize-list`
- `analysis/tools/local/fossilize/fossilize-disasm`
- `analysis/tools/local/rga/rga`
- `analysis/tools/local/bin/<tool>`
